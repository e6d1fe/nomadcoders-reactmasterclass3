### simple kanban to do list board

using:
- React
- TypeScript
- Recoil
- react-hook-form
- react-beautiful-dnd
- styled-components

you can:
- enter to do items in three pre-existing boards (to do, in progress, done)
- make a new board
- delete to do items you've already entered
- delete boards
- drag & drop to do items across and within boards
